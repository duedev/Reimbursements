Folders & Scheduled Export
Folder shortcuts

These paths drive the folder shortcut buttons and clipboard copy — they tell the app what the intake and output folders are called on your machine. To change which folder Docker actually uses: create a .env file next to docker-compose.yml with INTAKE_PATH=/your/path and restart the container.

Scheduled export

Automatically generate the spreadsheet into the export folder on a schedule. Point the export folder at a Dropbox / Google Drive / OneDrive sync folder for automatic cloud upload — or add a Dropbox token / enable email below.

Image Processing

Applied to every receipt as it's processed. Changes save automatically and take effect on the next receipt — no restart needed.
